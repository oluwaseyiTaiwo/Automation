This script is intended to scrape the network isis link on each router. 

Input Router "IP_ADDRESS" and "NAME" in the PULL_ISIS_NOKIA.csv.csv file provided
if PULL_ISIS_NOKIA.csv is deleted, create a new "CSV" file
name the column for ip address "Site ID" and the column for Router name "Name"

To run the script, enter into the directory of the PULL_ISIS_NOKIA.py file on CMD (Command Prompt)
type " python PULL_ISIS_NOKIA.py " to run the script
next, drag and drop CSV file containing the IP Address and ROuter name
